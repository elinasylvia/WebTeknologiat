Grid

CSS Grid layout on kaksiulotteinen taulukko, johon käyttöliittymän komponentit asetellaan.
Gridissä on container, jonka sisään sisältö eli “itemit” laitetaan. Asettelu tapahtuu kahdesssa ulottuvuudessa

Container

Jos haluat saada HTML-elementin käyttäytymään grid containerina, sinun on asetettava ominaisuudeksi display grid tai inline-grid.

Grid containerit koostuvat ruudukkoelementeistä, jotka on sijoitettu sarakkeisiin ja riveihin.

Grid-template-columns -ominaisuus
Grid-template-columns -ominaisuus määrittelee ruudukon asettelun sarakkeiden määrän ja se voi määrittää kunkin sarakkeen leveyden.

Arvo on välilyönnillä erotettu luettelo, jossa kukin arvo määrittää kyseisen sarakkeen leveyden.

Jos haluat, että ruudukkoasettelu sisältää neljä saraketta, määritä neljän sarakkeen leveys tai "auto", jos kaikilla sarakkeilla on oltava sama leveys.

Alueet

Grid-template-areas määrittää ruudukkomallin viittaamalla ruudukkoalueiden nimiin, jotka on määritetty ominaisuudella grid-area. Sisältö kattaa solut, jotka on nimetty samalla grid-arean nimellä. Piste tarkoittaa tyhjää solua.

Välit

Solujen väli voidaan määrittää seuraavilla: column-gap ja row-gap ominaisuuksilla.

.container { grid-template-columns: 100px 50px 100px; grid-template-rows: 80px auto 80px; column-gap: 10px; row-gap: 15px; }

Asettelu

Gridillä voi myös määrittää mihin solun sisältö asettuu pysty - ja vaakasuunnassa. Sisältö tulee asettaa joko jollekin alueelle tai johonkiin soluun/soluihin. Justify-items - kohdistaa ruudukon kohteet rivi-akselin mukaisesti. Tämä arvo koskee kaikkia säiliön sisällä olevia ruudukon kohteita.

start
kohdistaa elementit siten, että ne ovat samassa tasossa niiden solun aloitusreunan kanssa.
end
kohdistaa elementit siten, että ne ovat samassa tasossa niiden solun loppureunan kanssa.
center
kohdistaa kohteet solun keskelle.
stretch
täyttää solun koko leveyden (tämä on oletusarvo).
